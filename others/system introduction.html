Overview

XingGPT is a full-stack AI agent built on top of the Groq API (LLaMA 3, LLaVA, Whisper). It goes beyond a simple chat interface — the system automatically selects the right model for each query, maintains long-term memory across sessions, and can call external tools like live weather, a math evaluator, web search, and more. The frontend is a React (Vite) SPA; the backend is a lightweight Node.js / Express server acting as a secure middleware layer.

Core Features
Long-term Memory
Extracts user facts after each chat. Injected as system context on every future request.
Multimodal Vision
Attach images in chat. Routed to LLaVA vision model for analysis and Q&A.
Auto Model Routing
Backend inspects each message and picks the optimal model — no manual switching.
Tool Use
Weather, calculator, current time. Model calls tools autonomously via function calling.
Web Search
Real-time search via Tavily API. AI retrieves live data for current events and facts.
Audio Transcription
Upload audio or record live. Processed by Groq Whisper large-v3 for transcript.
